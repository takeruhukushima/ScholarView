ScholarViewは、研究者が執筆、公開、そして対話をシームレスに行えるよう設計された、デスクトップ級のワークスペースを提供します。

## 1. 3カラム・ワークスペース

効率的な研究ワークフローを支える、機能的なレイアウト構成です。

- **Sidebar (FileTree)**: 論文の各セクション（Markdown/TeX）や画像、BibTeXファイルを管理。ドラッグ＆ドロップによるインポートや移動をサポート。
- **Editor Panel (Workspace Editor)**: 論文の執筆と管理を行うメイン領域。
- **Right Panel (Discussion & Preview)**: 最終的なレンダリング結果の確認と、Bluesky上のリアルタイムな熟議スレッドを表示。

## 2. ハイブリッド・ブロックエディタ

ScholarViewのエディタは、MarkdownとLaTeXの表現力を兼ね備えた、独自のブロックベースの設計を採用しています。

- **TeX / Markdown 両対応**: KaTeXによる数式のリアルタイムレンダリングをサポート。
- **自動引用 (Citations)**: 執筆中に `\cite{}` や `[@citekey]` を入力すると、ワークスペース内のBibTeXファイルから自動的に候補を提示し、引用番号を割り当てます。
- **ドラッグ＆ドロップによるアセット管理**: 画像ファイルを直接エディタにドロップすると、自動的にPDSへアップロードされ、論文内にCIDで埋め込まれます。

## 3. Bluesky同期（Broadcast & Sync）

執筆した論文をワンクリックで学術コミュニティへ共有できます。

- **ブロードキャスト**: 論文をPDSに保存すると同時に、Blueskyへ「公開」を知らせるポストを自動投稿。
- **シンクロナイズ**: Bluesky上での反応（いいね、リポスト）をリアルタイムでワークスペース内に反映。
- **レコード管理**: 投稿済みの論文の更新や削除（Bluesky上のポストの削除を含む）を、エディタから直接操作可能。

## 4. インライン熟議 (Contextual Annotations)

論文の「特定箇所」に絞った、深い議論を可能にします。

- **ハイライト＆コメント**: 論文のプレビュー画面でテキストを選択すると、その箇所に対するBlueskyへのリプライを開始できます。
- **引用情報の保持**: どのテキストに対するコメントかを明示するため、Blueskyの外部リンクカードに引用コンテキストが自動的に埋め込まれます。
- **Bluesky上での見え方**: 通常のBlueskyアプリからは、論文への引用リンク付きのポストとして見え、ScholarView上からは該当箇所がハイライトされたインラインコメントとして展開されます。
